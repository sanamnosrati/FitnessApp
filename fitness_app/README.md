# BFit - Fitness App

A modern fitness app built with Flutter featuring workout tracking, nutrition guidance, and progress monitoring.

## Required Assets

### Illustrations
Place the following muscle group illustrations in `assets/illustrations/`:

- `upper_body.png` - Illustration showing chest, shoulders, back, and arm muscles
- `core.png` - Illustration showing abdominal and core muscles
- `lower_body.png` - Illustration showing leg and glute muscles
- `full_body.png` - Full body muscle illustration
- `cardio.png` - Illustration representing cardiovascular exercise

The illustrations should be PNG files with transparent backgrounds, ideally 512x512px or larger.

### Fonts
Download the Inter font family and place the following files in `assets/fonts/`:

- `Inter-Regular.ttf`
- `Inter-Medium.ttf`
- `Inter-SemiBold.ttf`
- `Inter-Bold.ttf`

You can download the Inter font family from [Google Fonts](https://fonts.google.com/specimen/Inter).

## Features

1. Workout Categories
   - Upper Body
   - Core
   - Lower Body
   - Full Body
   - Cardio

2. Nutrition
   - Recipe categories
   - Detailed nutritional information
   - Meal planning

3. Progress Tracking
   - Workout history
   - Body measurements
   - Achievement system

4. Customization
   - Personal goals
   - Workout preferences
   - Dietary restrictions

## Getting Started

1. Clone the repository
2. Add the required assets as described above
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app

## Theme

The app uses a luxurious gold and black theme:

- Primary Color: Rich gold (#FFD700)
- Secondary Color: Amber gold (#FFC107)
- Background: Deep black (#121212)
- Surface: Slightly lighter black (#1E1E1E)
- Text: White and light gray
