# Fitness App

A modern fitness and wellness mobile application developed with Flutter and Firebase.

## Features

- User authentication with Firebase
- Workout categories and exercise details
- Muscle-focused training plans
- Nutrition and calorie tracking
- Mental health and mood tracking
- Progress monitoring
- Responsive and modern UI
- Cross-platform support (iOS, Android, Web)

## Technologies Used

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Shared Preferences
- Google Sign-In
- Sign In with Apple
